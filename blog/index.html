Why is Deep Learning good?

Dec 03, 2020

Thanks to the David Sprott’s Distinguished Lecture by Professor Trevor Hastie and questions raised by Professor Ali Ghodsi in the Q&A session today, I realized that several things about Deep Learning. Why they are so good, I think there are three reasons:

  • For me, Deep Learning is just like a stack of linear models, so it is one way of ensemble models together with a certain structure, maybe hierarchical. Hence, the Deep Learning models usually contain more parameters than the classical models, which may explain why it can have better performance with bigger data. The data has been summarized by more parameters.
  • A lot of the model takes in knowledge about the data to give a good modeling. For example, the scan in CNN is just like a local kernel, which makes sense to capture the locally-correlated features in an image. Deep Learning also make use of data augmentation, just like we are adding a “gaussian cloud” to the data in classical modeling. These augmentation has more power because they are usually correct. In CNN, jittering the picture or rotating it we know that it still has the same label, i.e., we are adding in data with labels/response that we know is the true.
  • Deep Learning is online learning. It keeps updating with new data, i.e., we can use pre-train model for a new one. This makes it become very handy in a lot of applications.
